## Description

A nodeJs app to help track employees when needed. Users can update, remove and add employee, department and role data as needed via the command line using npm package inquirer.

#### Technologies Used

    	- Javascript
    	- NodeJS
    	- npm
    	- Inquirer package
    	- dotenv package
    	- mysql
    	- figlet package
    	- console.table package
